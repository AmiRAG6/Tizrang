🇺🇸 English

📌 About

TizRang is a powerful Bash-based cyber intelligence and defensive security analysis toolkit for Linux and Termux.

Designed for:

- Security Researchers
- Ethical Hackers
- Pentesters
- Linux Users
- Network Analysts
- Cybersecurity Learners

TizRang provides fast reconnaissance, domain intelligence, DNS analysis, SSL inspection, port scanning, risk analysis, and professional reporting — all inside a single-file architecture.

⚡ Installation

Clone Repository

git clone https://github.com/AmiRAG6/Tizrang.git
cd Tizrang

Install

bash TizRang.sh --install

---

🚀 Run

tizrang

---

🚀 Run without installation 

chmod +x TizRang.sh

./TizRang.sh

---

🗑 Uninstall

bash tizrang --uninstall

---

Key| Action
Ctrl + Q| Exit TizRang
